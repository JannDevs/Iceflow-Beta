Rules & Terms

Prohibited Actions

  • No Flooding/Spamming: Melebihi rate limit akan diblokir otomatis
  • No DDoS Attacks: Serangan akan diblokir secara otomatis
  • No Abuse: Jangan gunakan untuk hal yang merugikan
  • No Commercial Use: API ini gratis untuk personal use saja
  • No Reselling: Dilarang menjual kembali layanan ini

Usage Guidelines

  • Rate Limit: 10 requests per detik, 60 per menit, 100K per hari
  • Timeout: Request akan timeout setelah 30 detik
  • Fair Use: Gunakan secukupnya dan bijak
  • Attribution: Credit ke Iceflow API dihargai
  • Uptime: 99% uptime, maintenance berkala

Security & Privacy

  • Data Logging: Kami log IP, endpoint, dan timestamp untuk monitoring
  • No Personal Data: Kami tidak menyimpan data pribadi users
  • IP Blocking: IP yang melanggar akan diblokir otomatis
  • SSL/TLS: Semua request menggunakan HTTPS

Contact & Support

  • Issues: Laporkan bug atau masalah via Email
  • Feature Request: Saran fitur baru sangat diterima
  • Support: Response dalam 24–48 jam
  • Updates: Follow whatsapp channel untuk update terbaru
