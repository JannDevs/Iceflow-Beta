API Documentation

Learn how to integrate with our API using various programming languages

Basic API Request

curl -X GET "https://api.iceflow-beta.vercel.app/ai/koyuki?text=Hello"

With API Key

curl -X GET "https://api.iceflow-beta.vercel.app/ai/koyuki?text=Hello&apikey=YOUR_API_KEY"

Using requests library

import requests

url = "https://api.iceflow-beta.vercel.app/ai/koyuki"
params = {
    "text": "Hello",
    "apikey": "YOUR_API_KEY"
}

response = requests.get(url, params=params)
print(response.json())

Using Fetch API

const url = new URL("https://api.iceflow-beta.vercel.app/ai/koyuki");
url.searchParams.append("text", "Hello");
url.searchParams.append("apikey", "YOUR_API_KEY");

fetch(url)
    .then(response => response.json())
    .then(data => console.log(data));

Using file_get_contents

$url = "https://api.iceflow-beta.vercel.app/ai/koyuki?text=Hello&apikey=YOUR_API_KEY";
$response = file_get_contents($url);
$data = json_decode($response, true);
print_r($data);

Using Node.js with axios

const axios = require('axios');

axios.get('https://api.iceflow-beta.vercel.app/ai/koyuki', {
    params: {
        text: 'Hello',
        apikey: 'YOUR_API_KEY'
    }
})
.then(response => console.log(response.data));
