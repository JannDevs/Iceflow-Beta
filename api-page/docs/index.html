API Documentation

Learn how to integrate with our API using various programming languages

GET
/ai/koyuki?text={text}
Talk with Koyuki AI

Parameters

Parameter Type Required Description
text string Yes Text for Koyuki AI to respond to

Basic API Request

curl -X GET "https://api.iceflow-beta.vercel.app/ai/koyuki?text=Hello"

Example Response

{
  "status": "success",
  "creator": "Iceflow",
  "response": "Hello! How can I help you today?"
}

Using requests library

import requests

url = "https://api.iceflow-beta.vercel.app/ai/koyuki"
params = {
    "text": "Hello"
}

response = requests.get(url, params=params)
print(response.json())

Example Response

{
  "status": "success",
  "creator": "Iceflow",
  "response": "Hello! How can I help you today?"
}
